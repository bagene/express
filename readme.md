# SFIT API


API server for SAO/SFIT mostly used for Asana Tools


# Features

  - api docs



### Tech

Techs used:

* [NodeJS]
* [ExpressJS](https://expressjs.com/)
* [Swagger UI](https://swagger.io/tools/swagger-ui/)


### Installation

```sh
$ npm install
```

### Development

```sh
$ npm run dev
```


### Production

```sh
$ npm start
```

### PM2 

```sh
$ pm2 start pm2.json
```
